## Usage

```bash
  node index.js --query string [--platform google|bing] [--limit number]
```

### Examples

```bash
  node index.js --query pikachu
  node index.js --query pikachu --platform google
  node index.js --query pikachu --platform google --limit 10
  node index.js --query pikachu --limit 10
  node index.js --query "Ash Ketchum" --platform google
```
